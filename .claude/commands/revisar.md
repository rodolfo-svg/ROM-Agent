# /revisar - Revisão de Português Jurídico

Revisa e aprimora o texto jurídico com português escorreito e perfeito.

## Argumentos
- `$ARGUMENTS` - Texto para revisão ou caminho do arquivo

## Aspectos Revisados

### 1. Ortografia
- Correção ortográfica (Acordo Ortográfico vigente)
- Acentuação gráfica
- Hífen
- Uso de maiúsculas/minúsculas

### 2. Concordância
- Concordância verbal
- Concordância nominal
- Casos especiais de concordância

### 3. Regência
- Regência verbal
- Regência nominal
- Crase

### 4. Pontuação
- Vírgula
- Ponto e vírgula
- Dois-pontos
- Travessão
- Parênteses

### 5. Coesão e Coerência
- Conectivos adequados
- Progressão temática
- Referenciação
- Não contradição

### 6. Estilo Jurídico
- Formalidade adequada
- Clareza e precisão
- Objetividade
- Impessoalidade
- Evitar prolixidade

### 7. Vocabulário Jurídico
- Termos técnicos corretos
- Latinismos apropriados
- Expressões jurídicas consagradas

## Formato da Revisão

```
═══════════════════════════════════════════════════════════
REVISÃO DE PORTUGUÊS JURÍDICO
═══════════════════════════════════════════════════════════

ESTATÍSTICAS:
- Palavras: [XXXX]
- Sentenças: [XXX]
- Parágrafos: [XX]
- Correções realizadas: [XX]

CORREÇÕES POR CATEGORIA:
┌──────────────────────┬───────┐
│ Ortografia           │   XX  │
│ Concordância         │   XX  │
│ Regência             │   XX  │
│ Pontuação            │   XX  │
│ Estilo               │   XX  │
│ Vocabulário          │   XX  │
└──────────────────────┴───────┘

DETALHAMENTO DAS CORREÇÕES:

1. [ORIGINAL]: "texto original"
   [CORRIGIDO]: "texto corrigido"
   [MOTIVO]: Explicação da correção

2. ...

TEXTO REVISADO:
[Texto completo revisado]

SUGESTÕES DE APRIMORAMENTO:
[Sugestões adicionais para melhorar o texto]

═══════════════════════════════════════════════════════════
```
