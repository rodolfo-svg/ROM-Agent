## 🔧 TROUBLESHOOTING

### **Problema: "Permissão negada"**

```bash
chmod +x ~/Desktop/🤖\ ROM-Agent\ Opus.command
```

### **Problema: "Claude não encontrado"**

```bash
curl https://install.claude.ai | sh
```

Depois feche e reabra o Terminal.

### **Problema: "Diretório não encontrado"**

Verifique se ROM-Agent está em:
```bash
ls ~/Desktop/ROM-Agent
```

Se estiver em outro lugar, edite o atalho:
```bash
nano ~/Desktop/🤖\ ROM-Agent\ Opus.command
```

E mude a linha:
```bash
cd ~/Desktop/ROM-Agent
```

Para o caminho correto.

### **Problema: "GitHub não autenticado"**

```bash
gh auth login
```

### **Problema: "Render não autenticado"**

```bash
npm install -g render-cli
render login
```

## 📊 COMANDOS DO ATALHO

O atalho executa esta sequência:

```bash
# 1. Limpar terminal
clear

# 2. Mostrar banner
cat banner.txt

# 3. Ir para o projeto
cd ~/Desktop/ROM-Agent

# 4. Executar verificações
source .claude/init.sh

# 5. Criar prompt temporário
echo "$INIT_PROMPT" > /tmp/rom-agent-init-prompt.txt

# 6. Iniciar Claude Code com Opus
cat /tmp/rom-agent-init-prompt.txt | claude --model opus

# 7. Limpar e aguardar fechar
rm -f /tmp/rom-agent-init-prompt.txt
read -p "Pressione ENTER para fechar..."
```

## 💡 MODELOS DISPONÍVEIS

O atalho usa **Claude Opus 4.5** por padrão.

Para usar outros modelos, edite o atalho e mude:

```bash
claude --model opus    # Opus 4.5 (padrão)
claude --model sonnet  # Sonnet 4.5 (mais rápido)
claude --model haiku   # Haiku (mais econômico)
```

**Quando usar cada modelo:**

| Modelo | Quando usar |
|--------|-------------|
| **Opus** | Refatoração complexa, debugging difícil, arquitetura |
| **Sonnet** | Desenvolvimento geral, features, correções comuns |
| **Haiku** | Tarefas simples, testes, documentação |
